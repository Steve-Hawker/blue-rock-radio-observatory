## Signal Chain

Configured as: **Feed (Antenna + LNA1 + SAW1 + LNA2 + SAW2) → 6m LMR240 → SDR**

*Note: All active components (LNA1, SAW1, LNA2, SAW2) are within the feed enclosure
at the focal point. This is the optimal configuration — LNA1 is at the feed with
no cable loss preceding it.*

Signal chain confirmed 2026-05-07 via KrakenRF:

```
Antenna → QPL9547 (LNA1) → TA1077A (SAW1) → QPL9547 (LNA2) → TA2494A (SAW2) → Output
```

SAW filter part numbers (TA1077A / TA2494A) identified from PCB photo analysis
and datasheet matching — awaiting KrakenRF confirmation.
