### Pending hardware decision — Raspberry Pi 5

**Decision:** Replace Pi 3 + Pi 2 in DDOEE with a single Raspberry Pi 5 (4GB).

**Rationale:**
- Pi 3 has shared USB 2.0 hub — insufficient bandwidth for Airspy R2 + V4c simultaneously
- Two Pis require ethernet hub inside DDOEE or physical switching — adds complexity
- Pi 5 has USB 3.0, ~3× CPU performance, handles both SDRs simultaneously
- Single Pi, single ethernet connection — cleaner architecture
- Pi 3 and Pi 2 retained as spares

**Impact on documents when decision confirmed:**
- E001 → update system status
- DUAL_SDR_ARCHITECTURE.md → single Pi
- System architecture diagram → remove Pi 2, update Pi 3 to Pi 5
- RASPBERRY_PI_SETUP.md → Pi 5 specific notes
- ADF4351_CALIBRATION.md → update SPI reference from Pi 3 to Pi 5

**Timing:** Order before Phase 4 begins (dish assembly, August 2026).
Pi 3 adequate for Phase 2 (site survey) and Phase 3 (RFI commissioning).
