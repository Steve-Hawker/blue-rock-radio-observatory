## Purpose

This directory contains notes on papers, book chapters, and other
resources read in support of the Blue Rock Radio Observatory programme
and BSc Honours thesis.

---

## Workflow

**Read → write notes → add to Zotero**

Not the reverse. Resources are added to Zotero when notes are written —
not speculatively on discovery. This ensures the Zotero library
accurately reflects what has actually been engaged with rather than
what was intended to be read.

A notes file here is the trigger for a Zotero entry. If a resource
is worth writing notes about it is worth citing. If it is not worth
notes it probably does not belong in the bibliography.

---

## File naming convention

```
Author_Year_ShortTitle.md
```

Examples:
```
Chemin_2009_M31_HI_rotation.md
Baars_1977_flux_calibration.md
Wakker_2007_ComplexC_distance.md
BenBekhti_2016_HI4PI_survey.md
Clark_Clark_PracticalSDR_Ch2.md
Laufer_RTL-SDR_Guide_performance.md
```

For book chapters, use the chapter topic rather than chapter number —
chapter numbers change between editions, topics don't.

---

## Cross-references

Each notes file should reference the relevant programme documents:

- Learning plan topics (Topic 1–7)
- Design documents (design/*.md)
- Investigation documents (investigations/INV00X)
- Target files (targets/*.md)

This creates a web of connections between reading and programme
decisions — useful when writing the thesis methodology chapter.
